Purpose of the Program
The Best Buy Retail POS System is designed to automate the checkout process for a small retail store. It manages a product inventory, allows cashiers to build a customer's shopping cart, handles stock validation, calculates taxes and discounts, and generates a professional receipt.

How to Run the Program
1. Ensure Python 3.14 is installed on your machine.
2. Download the file: MajorProject.Group2-POS-ITT103-SP2026.py
3. Open a terminal or command prompt.
4. Navigate to the file.
5. Run the program
6. Follow the on-screen menu instructions.

Modifications & Features
Inventory Management: Uses a dictionary to store price and stock. 
Automatically deducts stock when items are added to the cart and returns stock 
if items are removed.

Discount Logic: Implemented a 5% discount for transactions exceeding $5000 
to encourage bulk purchases.

Validation: The system includes 'try-except' blocks to prevent the program 
from crashing if a user enters text instead of a number for prices or quantities.

Stock Validation: Logic was added to "add_to_cart" function to prevent customers from adding more items than what is in stock.

Receipt Generation: The "generate_receipt" function is used to provide a clear breakdown of the transaction.




Assumptions & Limitations
- Assumption: All prices are in the USD ($).
- Assumption: The tax rate is fixed at 10%.
- Limitation: The inventory is reset to its default values every time the 
  program is restarted (no file storage).
- Limitation: Product names must be typed correctly even though the system 
  is case-insensitive.
